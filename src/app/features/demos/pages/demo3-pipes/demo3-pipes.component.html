Les pipes

Pipe intégrés

Il existe de nombreuse pipe intégrées à Angular. Voici quelques exemples :

  • uppercase : {{ "hello world" | uppercase }}
  • lowercase : {{ "HELLO WORLD" | lowercase }}
  • titlecase : {{ "hello world" | titlecase }}
  • date : {{ uneDate | date }}
  • date short : {{ uneDate | date : "short" }}
  • date shortDate: {{ uneDate | date : "shortDate" }}
  • date shortTime: {{ uneDate | date : "shortTime" }}
  • date medium: {{ uneDate | date : "medium" }}
  • date avec format: {{ uneDate | date : "dd/MM/yyyy" }}
  • currency : {{ 100 | currency }}
  • currency : {{ 100 | currency : "EUR" }}
  • nombre : {{ 0.533 | number }}
  • percent : {{ 0.533 | percent }}
  • nombre avec minimum de chiffre affiché : {{ 0.533 | number : "2.2" }}
  • nombre float maximum après la virgule : {{ 0.533 | number : "0.0-2" }}
  • nombre avec minimum de chiffre affiché et séparateur de millier : {{ 1000 | number : "2.2-2" }}
  • json : {{ { name: "John", age: 30 } | json }}
  • slice (hello world, de 1 à 7): {{ "hello world" | slice : 1 : 7 }}
